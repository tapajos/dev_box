My development box.

## Details

* VirtualBox and VMWare support
* Ubuntu precise 64
* Ruby (last stable version)
* MySQL
* Memcached
* Redis
* Erlang and Elixir
* My dot files repo

## Dependencies

* omnibus.

		vagrant plugin install vagrant-omnibus

## Usage

		vagrant up
